1. Package Overview

The CD-ROM contains the software programs for the Sena Parani-UD100 product and the related documentation files. 
The following files are included in this CD-ROM package.

- AutoRun
	+ Flash-based user interface files for easier use of the Parani-UD100_CDROM.
- Parani_UD100
	+ Documents and utilities for the Sena Parani-UD100 product.
	+ Documents include Data sheet, Quick start guide and the Users' manual either in English and Korean version.
- Install Files
	+ Installation execution files for Bluetooth Software.
- htm
	+ contact us html files either in English and Korean version.

2. Folder structure

/Parani_UD100

	Datasheet, Quick Start Guide and the Users' guide manual of the Parani-UD100
		
	
/Install Files
	Installation execution files for Bluetooth Software.	
			

/htm
	contact us html files either in English and Korean version.
	/img
		Image files for html files
	

		
3. Document file naming convention

	Type_ProductModelName_Language-Version.pdf

	Type: qsg (Quick Start Guide), ds (Datasheet), manual (Users' Guide Manual)
	ProductModelName: Parani-UD100
	Language:English by default

- Qucik Start Guide 	(e.g. qsg_ud100_eng.pdf)
- Data sheet 		(e.g. ds_parani_ud100.pdf)
- Users' guide manual 	(e.g. manual_parani_ud100-v3.0.0.pdf)
